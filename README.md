# XEasyWork

#### 介绍
基于tinyflow-ai的完整应用项目

#### 软件架构
软件架构说明


#### 安装教程

1.  导入sql
2.  修改后端yml配置
3.  将tinyflow-java-core最新代码打包，放入maven仓库（https://gitee.com/tinyflow-ai/tinyflow-java）
4.  cd xeasywork-vue
5.  npm i
6.  npm run dev

#### 更新教程
1.  新建数据库导入sql文件
2.  使用navicat数据库工具进行表结构同步
3.  进行数据同步-菜单表（sys_menu）、字典表数据（sys_dict_data、sys_dict_type）（其他表可选）
